# ArielsSequitur
Implementation of the sequitur algorithm following the Craignm Ideas (C++).
At running this implementation prints the output grammar an some stats.

## To compile
- On the repo directory
```
$ make
```

## To run tests
- compile
- On the repo directory
```
./TestSequitur
```

## To run sequitur
- compile
- run tests
- On the repo directory
  - From stdin
```
./Sequitur
```
   - From file (:B)
```
cat <filename> | ./Sequitur
```
